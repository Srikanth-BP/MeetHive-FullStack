# MeetHive

This project was developed and manually tested during my MCA academic coursework.

MeetHive is a full-stack web application that allows users to interact through posts and basic social features.

## Tech Stack
Frontend: HTML, CSS, TypeScript  
Backend: Java / Spring Boot  
Database: MySQL  
API Testing Tool: Postman

## Features
- User registration and login
- Create and manage posts
- Backend REST API integration
- Database connectivity

## Testing
Manual testing was performed after development to verify functionality.

Testing activities included:
- Functional testing of user flows
- Validation of backend APIs using Postman
- Regression testing after feature updates
- Verification of application responses and user interactions

## Project Structure
/Frontend_Meethive
Contains the UI components and client-side code.

/Backend_Meethive
Contains server-side logic, APIs, and database integration.
